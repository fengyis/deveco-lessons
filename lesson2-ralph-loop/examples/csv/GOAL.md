# 目标
仓库里已经有一个 CSV 单行解析器 `src/csv.ts` 和它的测试 `test/csv.test.ts`。
现在 `bun test` 是**红的**——天真实现只会按逗号硬切,不认引号。
你的活是**把它修对**,让所有测试通过。

`parseLine(line: string): string[]` 的规则:

- 字段用 `,` 分隔:`"a,b,c"` → `["a","b","c"]`
- 字段可以被双引号包裹,**引号内的逗号是字面量**:`'"a,b",c'` → `["a,b","c"]`
- 引号内的 `""` 表示**一个字面双引号**:`'"she said ""hi"""'` → `['she said "hi"']`
- **空字段必须保留**(开头 / 中间 / 结尾 / 空引号都算一个字段):
  `"a,,c"` → `["a","","c"]`;`",a"` → `["","a"]`;`"a,"` → `["a",""]`;`'"",x'` → `["","x"]`
